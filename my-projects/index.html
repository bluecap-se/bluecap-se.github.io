Proghetto

Jun, 2008

This web application was built by me and four other as a university project in system development. It is a project managing system for multiple users. Among many things, it contains a calendar, gantt scheme, file management and statistics.

We were given the task to create something different, and since the entire group consisted of non-swedes - we decided to go ghetto-style. The nameProghetto- for example - comes from the italian wordprogetto, which meansproject. Both the language and design is influensed by the ghetto style, while still maintaining the funcionality required for the task.

Proghetto was built with a backend containing PHP and MySQL. Frontend contains XHTML 1.0, CSS and Javascript. The design was created in Photoshop CS by me.
